# Biopulze. version 1

BioPluze es una aplicación web híbrida desarrollada en Apache Cordova, orientada a la gestión, mantenimiento y calibración de equipos biomédicos. Su funcionamiento se basa en una arquitectura con servidor desplegado en una máquina virtual con Linux y una base de datos MySQL, permitiendo la centralización de la información y el control eficiente de los procesos técnicos. La aplicación facilita el registro, seguimiento y administración de actividades relacionadas con los equipos, con el fin de optimizar la gestión, garantizar el cumplimiento de mantenimientos y mejorar la trazabilidad dentro del entorno biomédico.
